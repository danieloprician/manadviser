# Technical Documentation

Documentația tehnică pentru dezvoltarea și mentenanța ManAdviser Insurance Platform.

## Available în viitor

- **Architecture Overview** - Diagrame de arhitectură sistem
- **API Documentation** - Documentația completă a API-ului .NET
- **Database Schema** - Schema bazei de date și migrări  
- **Development Setup** - Ghid pentru setup-ul mediului de development
- **Deployment Guide** - Ghiduri pentru deployment în producție
- **Security Guidelines** - Măsuri de securitate și best practices
- **Testing Strategy** - Strategia de testare (unit, integration, e2e)
- **Performance Monitoring** - Monitoring și optimizări de performanță
- **Troubleshooting** - Ghid pentru rezolvarea problemelor comune

## Stack actual
- **Frontend**: React 18 + Vite + Tailwind CSS
- **Backend**: ASP.NET Core 8 + Entity Framework
- **Database**: SQL Server (LocalDB) / PostgreSQL
- **Authentication**: JWT
- **Languages**: Bilingual (RO/EN) cu react-i18next
